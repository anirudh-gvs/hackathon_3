Core Capabilities

🧠

Local LLM Inference

Powered by GGUF-quantized Phi-3-mini running directly on CPU. Fully private structured data extraction with zero external API calls.

🔍

Hybrid OCR Pipeline

Integrates Pytesseract for images and PDFPlumber for digital documents to output clean, raw text ready for the parser engine.

📁

Predefined Schemas

Extract data matching specific Pydantic schemas: Invoices/Receipts (vendor, total, tax), Medical Reports, or custom Generic structures.
