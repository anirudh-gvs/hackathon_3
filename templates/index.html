About Offline DocScan

Privacy-first, offline AI document processing

Offline DocScan is a CPU-first, offline-only document processing application developed for the CPU First Hackathon. All processing happens locally on your device - no data leaves your computer.

Features

  • Extract structured data from PDFs, images, and text files
  • Support for Receipt, Medical, and Generic document schemas
  • 100% offline - no internet required
  • CPU-only processing - no GPU needed
  • Local storage of scan history

Technology Stack

  • Python Flask backend
  • pdfplumber for PDF text extraction
  • Tesseract OCR for images
  • llama-cpp-python for local LLM inference
  • Pydantic for schema validation
